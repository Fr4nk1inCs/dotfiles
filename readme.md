# My Dotfiles

```plaintext
config/*     -> ~/.config/*
local/*      -> ~/.local/*
fonts/*      -> ~/.fonts/*      # Only *.ttf and *.otf needed
clang-format -> ~/.clang-format
zimrc        -> ~/.zimrc        # Not sure how it works
zshrc        -> ~/.zshrc
```

## Installation

```bash
git clone https://github.com/Fr4nk1inCs/dotfiles.git
cd dotfiles

# Install fonts first
cp fonts/Feather.ttf ~/.fonts/
cp fonts/SFMono-Nerd-Font-Ligaturized/*.otf ~/.fonts/
fc-cache -fv

# Copy the configuration files you need.
cp -r config/* ~/.config/
cp -r local/* ~/.local/
```
